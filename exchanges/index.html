Fetch.fetch_ohlcv — Method

Fetch ohlcv data from exchange for a list of pairs.

  • from, to: Can represent a date. A negative from number implies fetching the last N=from candles.
  • update: If true, will check for cached data, and fetch only missing candles. (false)
  • progress: if true, show a progress bar. (true)
  • reset: if true, will remove cached data before fetching. (false)
source
Fetch.fetch_ohlcv — Method

Fetch ohlcv data for multiple exchanges on the same timeframe.

It accepts:

  • a mapping of exchange instances to pairlists.
  • a vector of symbols for which an exchange instance will be instantiated for each element, and pairlist will be composed according to quote currency and min_volume from PingPong.config.
